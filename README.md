🏫 School Management Suite

A complete school management platform that simplifies administrative tasks for educational institutions.

✨ Features:
• 👤 User Management (Students, Teachers, Admins)
• 📚 Class & Subject Management
• 📊 Grade & Exam Tracking
• 📝 Attendance Monitoring
• 💰 Fee & Payment Processing
• 📅 Timetable Management
• 🏠 Hostel & Transport Management
• 📢 Announcements & Calendar
• 📈 Reports & Analytics

Built with: Next.js 14, Appwrite, TypeScript, Tailwind CSS
